\begin{problem}{$Z$-функция}{z.in}{z.out}{1 секунда}{64 мегабайта}

Дана строка $S$. Необходимо вычислить $Z$-функцию для всех её позиций.

\InputFile

Во входном файле записана единственная строка $S$, состоящая из прописеных латинских 
букв($1 \le |S| \le 1000000$).

\OutputFile
Выведите через пробел  $Z(2), Z(3), \dots, Z(|S|)$.

\Example

\begin{example}
\exmp{
ababcbc
}{
0 2 0 0 0 0
}%
\exmp{
aaaaaa
}{
5 4 3 2 1
}%
\end{example}

\end{problem}
