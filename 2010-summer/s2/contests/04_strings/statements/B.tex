\begin{problem}{$\pi$-функция}{pi.in}{pi.out}{1 секунда}{64 мегабайта}

Дана строка $S$. Необходимо вычислить $\pi$-функцию для всех её начал.

\InputFile

Во входном файле записана единственная строка $S$, состоящая из прописеных латинских 
букв($1 \le |S| \le 1000000$).

\OutputFile
Выведите через пробел  $\pi(1), \pi(2), \dots, \pi(|S|)$.

\Example

\begin{example}
\exmp{
ababcbc
}{
0 0 1 2 0 0 0
}%
\exmp{
aaaaaa
}{
0 1 2 3 4 5
}%
\end{example}

\end{problem}
