\begin{problem}{На клетчатой бумаге}{\name.in}{\name.out}{}{}

На клетчатой бумаге нарисована окружность целого радиуса $R$ с центром на пересечении линий. 
Найти $K$ - количество клеток, целиком лежащих внутри этой окружности.

\InputFile

Первая строка входного файла содержит число $R$ ($1 \le R \le 10000$).

\OutputFile

Ваша программа должна вывести число $K$.

\end{example}

\end{problem}
