\def\name{div}%
\begin{problem}{A div B}{\name.in}{\name.out}{}{}

Вычислите частное (целую часть) при делении $A$ на $B$.

\InputFile

Первая строка входного файла содержит целое число $A$, вторая - целое число $B$
($0 \le A,B \le 10^{3000}$).

\OutputFile

Первая строка выходного файла должна содержать одно целое число - $A div B$.

\Examples

\begin{example}
\exmp{
66350
896
}{
74
}%
\exmp{
12
13
}{
0
}%
\exmp{
66151802969158216
63962
}{
1034235999017
}%
\end{example}

\end{problem}
