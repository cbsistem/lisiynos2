\begin{problem}{„исла в вершинах}
{vnums.in}{vnums.out}
{2 секунды}{256 мегабайт}

¬ неориентированном графе без кратных ребер и петель
расставить в вершинах числа так, чтобы если вершины
соединены ребром, то числа имели общий делитель, а если нет - то нет.

\InputFile

¬ файле записано число $N$ ($0<N<7$) - количество вершин в графе.
«атем записана матрица смежности.                                                                     

\OutputFile

¬ файл вывести $N$ натуральных чисел из диапазона $[1,2147483647]$,
которые вы предлагаете приписать вершинам.

\Example

\begin{example}
\exmp{%
3
0 1 1
1 0 0
1 0 0
}{%
6 2 3
}%
\end{example}

\end{problem}
