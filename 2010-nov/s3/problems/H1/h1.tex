\def\name{h1}%
\begin{problem}{Списки}{\name.in}{\name.out}{}{}

Дан список. 
Нужно эффективно реализовать операции добавления в него элемента и удаления элемента из него.
Изначально список пустой.

\InputFile

Входной файл содержит на каждой строке по одной из команд:
\begin{itemize}
  \item \t{INSERT} <<значение элемента>>       - Добавление в конец.
  \item \t{DELETE} <<номер элемента c начала>> - Удаление из начала.
\end{itemize}



\OutputFile

В выходной файл нужно вывести удаленные элементы, по одному на каждой строке.

\Examples

\begin{example}
\exmp{ \input \name/01 }{ \input \name/01.a }% 
\end{example}

\end{problem}
