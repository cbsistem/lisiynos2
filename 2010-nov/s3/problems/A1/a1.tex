\def\name{a1}%
\begin{problem}{Строки в книге}{\name.in}{\name.out}{}{}


В книге на одной странице помещается $K$ строк. 
Таким образом, на 1-й странице печатаются строки с 1-й по $K$-ю, на второй - с $(K+1)$-й по $(2K)$-ю и т.д. 
Напишите программу, которая по номеру строки в тексте определяет номер страницы, на которой будет напечатана эта строка, и порядковый номер этой строки на странице.

\InputFile

Входной файл содержит число $K$ - количество строк, которое печатается на странице, и число $N$ - номер строки 
($1 \le K \le 200$, $1 \le N \le 20000$).

\OutputFile

В выходной файл выведите два числа - номер страницы, на которой будет напечатана эта строка и номер строки на странице.

\Examples

\begin{example}
\exmp{ \input \name/01 }{ \input \name/01.a }% 
\exmp{ \input \name/02 }{ \input \name/02.a }% 
\exmp{ \input \name/03 }{ \input \name/03.a }% 
\exmp{ \input \name/04 }{ \input \name/04.a }% 
\exmp{ \input \name/05 }{ \input \name/05.a }% 
\exmp{ \input \name/06 }{ \input \name/06.a }% 
\end{example}

\end{problem}
