\begin{problem}{A-B}{\name.in}{\name.out}{}{}

Вычислите разность $A-B$.

\InputFile

Первая строка входного файла содержит целое число $A$, вторая - целое число $B$
($0 \le A,B \le 10^{3000}$).

\OutputFile

Первая строка выходного файла должна содержать целое число - $A-B$. Ответ может быть отрицательным!

\Examples

\begin{example}
\exmp{
6
495
}{
-489
}%
\exmp{
812100766
641888
}{
811458878
}%
\end{example}

\end{problem}
