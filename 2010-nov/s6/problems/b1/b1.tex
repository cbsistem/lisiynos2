\begin{problem}{Наибольший общий делитель}{\name.in}{\name.out}{}{}

Вычислите наибольший общий делитель массива из $N$ целых чисел.

\InputFile

Первая строка входного файла содержит целое число $N$ - количество натуральных чисел ($1 \le N \le 10000$).
Вторая - $N$ натуральных чисел ($1 \le A_i \le 2^{63}-1$).

\OutputFile

Первая строка выходного файла должна содержать одно целое число - НОД $A_1,A_2,...,A_N$.

\Examples

\begin{example}
\exmp{ \input \name/01 }{ \input \name/01.a }% 
\exmp{ \input \name/02 }{ \input \name/02.a }% 
\exmp{ \input \name/03 }{ \input \name/03.a }% 
\end{example}

\end{problem}
