\begin{problem}{Расстояние}
{dist.in}{dist.out}{2 секунды}{64 мегабайта}

  На плоскости заданы точка и прямая. Требуется найти расстояние между ними.

  Прямая задана уравнением $Ax+By+C=0$. Точка задана координатами $(x_0;y_0)$.


\InputFile

\def\nohyphenation{\hyphenchar\font = -1  \aftergroup\restorehyphenation}
\def\restorehyphenation{\hyphenchar\font = `- }

 Входной файл содержит несколько (до 1000) тестов. 
 Каждый тест записан в отдельной строке и содержит 5 чисел $A$,$B$,$C$,$x_0$,$y_0$ разделённых пробелами. 
 Все числа целые и не превосходят по модулю 10000.
 Окончание тестов --- "0 0 0"  на отдельной строке.

\OutputFile

 Для каждого теста в выходной файл одно число --- расстояние с точностью до 3-х знаков после точки.
 
\Examples

\begin{example}
\exmp{
0 1 2 2 3
4 0 0 6 4
2 1 3 7 6
0 0 0
}{
5.000
6.000
10.286
}%
\end{example}

\end{problem}
