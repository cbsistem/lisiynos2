\begin{problem}{Параллельность}
{par.in}{par.out}{2 секунды}{64 мегабайта}

  На плоскости заданы две прямых. 
  Gfhfkktkmys kb jyb?

  Прямые заданы уравнениями $Ax+By+C=0$. 


\InputFile

\def\nohyphenation{\hyphenchar\font = -1  \aftergroup\restorehyphenation}
\def\restorehyphenation{\hyphenchar\font = `- }

 Входной файл содержит несколько (до 1000) тестов. 
 Каждый тест занимает 2 строки, в каждой по 3 числа: $A_1$,$B_1$,$C_1$,
 и $A_2$,$B_2$,$C_2$,
 Все числа целые и не превосходят по модулю $10^4$.
 Окончание тестов --- "0"  на отдельной строке.

\OutputFile

 Для каждого теста в выведите ``\t{YES}'', прямые параллельны, 
      и ``\t{NO}'' в противном случае.
\Examples

\begin{example}
\exmp{
0 1 2 2 3
4 0 0 6 4
2 1 3 7 6
0 0 0
}{
5.000
6.000
10.286
}%
\end{example}

\end{problem}
